# Sistemas Operativos - fundamentos, apuntes
## ¿Qué es?
> Un sistema operativo (SO) es el conjunto de programas de un sistema informático que gestiona los recursos del hardware y provee servicios a los programas de aplicación de software. En últimas, intermediario entre hardware, software e individuo.
